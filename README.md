# Projet BTS SIO - SLAM

## Description
Application web développée avec Node.js, Express et MySQL.

## Fonctionnalités
- Gestion des utilisateurs
- API REST
- Connexion à une base de données

## Technologies utilisées
- JavaScript
- Node.js / Express
- MySQL
- Sequelize
- HTML / CSS / Bootstrap

## Installation

1. Cloner le projet :
git clone [https://github.com/LisaRupert03/projet1-bts-sio]

2. Installer les dépendances :
npm install

3. Lancer le serveur :
npm start
